# ics_sandbox — Sensor Simulation & Validation Package

Python simulation models and pytest test suite for the five sensors in the ICS Core downhole package.

These files are **generated by `ICS_Core_Firmware0_Sensor IC_Assembly_Validation.ipynb`** via `%%writefile` cells. The notebook is the source of truth — edit models there, not directly here.

---

## Package Structure

```
ics_sandbox/
├── simulation/
│   └── models/
│       ├── __init__.py
│       ├── adxl206_model.py      # Dual-axis MEMS accelerometer
│       ├── adxrs645_model.py     # MEMS rate gyroscope
│       ├── at10tb_model.py       # IEPE vibration accelerometer
│       ├── lm95172_mock.py       # SPI digital temperature sensor
│       └── max31856_mock.py      # SPI thermocouple converter
└── tests/
    ├── __init__.py
    ├── test_adxl206.py
    ├── test_adxrs645.py
    ├── test_at10tb.py
    ├── test_lm95172.py
    └── test_max31856.py
```

---

## Running Tests

From the `ics_sandbox/` directory:

```bash
cd ics_sandbox
pytest tests/ -v
```

Or from the repo root:

```bash
pytest ics_sandbox/tests/ -v
```

## Sensor Models

### `adxl206_model.py` — ADXL206HDZ Dual-Axis Accelerometer

Used for inclination measurement (static tilt, not dynamic vibration).

| Parameter | Value |
|---|---|
| Sensitivity | 328 mV/g at 5 V |
| Range | ±5 g |
| Max temp | 175°C |
| Temp coefficient | −0.01%/°C on sensitivity |

Key functions:
- `acceleration_to_voltage(accel_g, temp_c, noise)` — converts g to output voltage
- `inclination_from_dual_axis(x_g, y_g)` — returns pitch/roll in degrees
- `simulate_static_tilt(pitch_deg, roll_deg, temp_c)` — forward model for known tilt

---

### `adxrs645_model.py` — ADXRS645HDYZ Rate Gyroscope

Used for RPM measurement and drill string rotation tracking.

| Parameter | Value |
|---|---|
| Sensitivity | 12.5 mV/dps |
| Range | ±2000 dps |
| Max temp | 175°C |
| Temp coefficient of null | 0.015 dps/°C |

Key functions:
- `rate_to_voltage(rate_dps, temp_c, noise)` — converts angular rate to output voltage
- `voltage_to_rate(voltage, temp_c)` — inverse: voltage to dps
- `dps_to_rpm(rate_dps)` — converts dps to RPM (÷ 6)

---

### `at10tb_model.py` — AT/10/TB IEPE Accelerometer

High-range vibration sensor. Full signal chain simulation including ICP source, AA filter, and 12-bit ADC.

| Parameter | Value |
|---|---|
| Sensitivity | 10 mV/g |
| Range | ±500 g |
| Sample rate | 50 kSa/s |
| AA filter | 4.8 kHz Butterworth |
| ADC | 12-bit, 3.3 V ref |
| DC offset | 1.65 V (signal centred in ADC window) |

Key functions:
- `simulate_vibration(accel_g_array, fs, temp_c)` — full chain: filter → offset → ADC → counts
- `adc_counts_to_g(counts)` — convert ADC counts back to g

---

### `lm95172_mock.py` — LM95172 SPI Temperature Sensor

Simulates the SPI register map for firmware driver testing. Used as the temperature reference for all sensor T-compensation.

| Parameter | Value |
|---|---|
| Range | −40 to 200°C |
| Resolution | 13–16 bit selectable |
| Interface | SPI 3-wire |

Key methods on `LM95172Mock`:
- `read_register(addr)` — returns register bytes matching datasheet format
- `set_temperature(temp_c)` — set the simulated ambient temperature
- `set_resolution(bits)` — 13/14/15/16-bit mode

---

### `max31856_mock.py` — MAX31856 Thermocouple Converter

Simulates Type K thermocouple interface for external formation temperature.

| Parameter | Value |
|---|---|
| TC range | −270 to 1800°C |
| CJC range | −40 to 125°C |
| Resolution | 0.0078125°C (19-bit) |
| Interface | SPI 4-wire |

Key methods on `MAX31856Mock`:
- `read_register(addr)` — register read with datasheet-accurate encoding
- `set_tc_temperature(temp_c)` — set thermocouple junction temperature
- `set_cj_temperature(temp_c)` — set cold-junction temperature
